# Section: Ask / Interact
 
%Short: Player asks a character questions, selects topics, and makes choices
 
## Included Ingredients
- Choice: Topic List
- Choice: Select Topic
- Choice: Forced Choice

- Line: Speech
- Line: Thought
- Line: Narration

- Line: Yes/No/Maybe (Choices)

- Story Beat: Text Someone (for text-based interactions)
- Story Beat: Call (for phone call interactions)
- Story Beat: Someone Calls Instead of Texting (for phone call interactions)
- Story Beat: Speak To Someone (for spoken dialogue interactions)

## Description
This section encompasses ingredients that facilitate player interactions with characters through questioning, topic selection, and making choices. It includes mechanisms for presenting topics, handling forced choices, and managing various forms of character communication such as texting, phone calls, and spoken dialogue.
**Input**: Character profiles, list of topics, dialogue lines, narrative text
**Process**: Player selects character to ask → chooses topic or question → engages in dialogue through text, call, or spoken interaction → makes choices as prompted. Player: repeats as wanted for each topics.
**Output**: Dialogue lines displayed, character states updated, events or flags modified

## Composition
- Select Character to Ask → choose which character to interact with
- Question Character → present a question to the character
- Topic List → provide a selectable list of discussion topics
- Select Topic → player chooses which topic to explore
- Yes / No / Maybe → branching responses or follow-ups
- Optional Story Beats → e.g., “Text Someone,” “Call Instead,” “Realization,” or “Reaction”

# Considerations
- May overload the story compared to linear dialogue
