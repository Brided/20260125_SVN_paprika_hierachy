# Choice: Topic List

@tag: Choice_Topic_List

%Short: A list of topics for the player to choose from in a conversation.

## Description

This unit provides a predefined list of topics that the player can select during a conversational interaction. Each topic serves as a prompt for dialogue between characters, allowing for varied interactions based on the player's choice.
**Input**: List of topics
**Process**: Player selects a topic → triggers associated dialogue branch → optional updates to character state or flags
**Output**: Dialogue lines displayed, events or flags modified

There are two types: regular topics and character-specific topics

## Notes
- Topics get added as the player inspects the environment or interacts with characters
