Expression Modes

# Line: Speech

%Short: Character speaks a line of dialogue, spoken or texted

## Description
This ingredient represents a single line of dialogue spoken or texted by a character. It is used within various story beats to convey what a character says during interactions, whether through spoken conversation, texting, or phone calls.
**Input**: Character profile, dialogue text
**Process**: Retrieve character information → Format dialogue line with character's name and text → Display line in the appropriate context (spoken, texted, etc.)
**Output**: Formatted dialogue line displayed to the player



# Line: Thought

%Short: Character's internal thought

## Description
This ingredient represents a single line of internal thought from a character. It is used to convey what a character is thinking during various story beats, providing insight into their feelings, motivations, or reactions that are not spoken aloud.
**Input**: Character profile, thought text
**Process**: Retrieve character information → Format thought line with character's name and thought text → Display line in the appropriate context (internal monologue)
**Output**: Formatted thought line displayed to the player



# Line: Narration

%Short: POV char describes what they do.

## Description
This ingredient represents a single line of narration from the point of view of a character. It is used to convey actions, settings, or events as perceived by the character, providing context and depth to the story.
**Input**: Character profile, narration text
**Process**: Retrieve character information → Format narration line with character's name and narration text → Display line in the appropriate context (narrative description)
**Output**: Formatted narration line displayed to the player
