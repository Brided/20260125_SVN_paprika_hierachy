@tag: StoryBeat_TextSomeone

% Type: Action

## Description
This unit represents a story beat focused on text-based interactions between characters. It includes dialogue exchanges, character thoughts, and narrative descriptions conveyed through text. This ingredient is essential for visual novels and interactive fiction where storytelling relies heavily on written content.
**Input**: Character profiles, dialogue lines, narrative text
**Process**: Display text on screen phone → Manage dialogue flow → Handle character interactions
**Output**: Text displayed to the player, character states updated

## Notes
- Can be used independently or as part of larger story structures
- Supports branching dialogue and multiple character interactions
- Can include internal monologues or narrative descriptions alongside dialogue
