# Story Beat: Call

@tag: StoryBeat_Call

% Type: Action
%Short: Characters engage in phone call interactions

## Description
This unit represents a story beat focused on phone call interactions between characters. It includes dialogue exchanges, character thoughts, and narrative descriptions conveyed through spoken words during a phone call. This ingredient is essential for visual novels and interactive fiction where storytelling relies on verbal communication between characters over the phone.
**Input**: Character profiles, dialogue lines, narrative text
**Process**: Display phone call dialogue on screen → Manage dialogue flow → Handle character interactions
**Output**: Phone call dialogue displayed to the player, character states updated

## Notes
- Split screen can be used to show both characters during the call
